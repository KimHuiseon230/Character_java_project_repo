3. 입력이 성공적으로 되었을 경우 '삽입성공' 출력

## 수정(update)
1. 정규표현식을 통한 입력값 제어
2. 삭제할 id 값을 먼저 입력 받고 해당 id가 존재하지 않으면 다시 값 입력 유도
3. 생존력, 힘, 지능 3개의 영역의 정보를 수정
4. 입력 값이 100이 넘어갈 경우 다시 입력하라는 오류 문구 출력
3. 입력된 데이터를 확인할수 있는 출력문구 확인
